
### docker-compose commands
https://docs.docker.com/compose/reference/


```bash
# Constrói, inicia e anexa contêineres para um serviço
docker-compose up -d

# Interrompe contêineres e remove contêineres, redes, volumes e imagens criados no docker-compose up
docker-compose down

# Inicia contêineres existentes para um serviço
docker-compose start

# Interrompe a execução de contêineres sem removê-los
docker-compose stop

# Reinicia contêineres existentes para um serviço
docker-compose restart

# Pausa a execução de contêineres de um serviço
docker-compose pause

# Reativa os contêineres pausados ​​de um serviço
docker-compose unpause

# Remove contêineres de serviço parados
docker-compose rm

# Lista contêineres
docker-compose ps
```
